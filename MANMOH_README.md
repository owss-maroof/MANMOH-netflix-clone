# MANMOH — Streaming Platform Clone

A Netflix-inspired streaming platform homepage built with pure HTML and CSS. No frameworks, no libraries — just clean front-end code.

## Live Demo

👉 [https://owss-maroof.github.io/MANMOH](https://owss-maroof.github.io/MANMOH)

## Preview

![MANMOH Homepage](screenshot.png)

---

## What It Includes

- Responsive header with language selector and sign-in button
- Full-screen hero section with background image and email CTA
- Horizontally scrollable movie row with hover zoom effects
- Contact Us section with form and info
- Clean dark theme throughout

| Technology | Usage |

## Features

**Header**
- Fixed navigation with language dropdown
- Sign In button with hover effect
- Custom MANMOH logo in Netflix-style red

**Hero Section**
- Full viewport background image
- Dark gradient overlay for readability
- Email input with Get Started CTA button

**Movie Row**
- Horizontally scrollable with hidden scrollbar
- Smooth zoom and brightness effect on hover
- Movie title displayed below each poster

**Contact Section**
- Name, email, and message fields
- Send Message button
- Contact info (email, phone, location)

---

## How to Run Locally

Just open `index.html` in any browser — no server needed.

```bash
git clone https://github.com/owss-maroof/MANMOH.git
cd MANMOH
# open index.html in browser
```
